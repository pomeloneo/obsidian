🔥 本周热门

**React 基金会成立** —— 这几年，React 的“归属权”和“控制权”一直是社区讨论的焦点，大家总担心如果哪天发生变动会怎么办。在 React Conf 2025 上，官方宣布 React 及 React Native 的管控权将从 Meta 交接到一个独立的基金会，初期成员包括 Amazon、Expo、Meta 和 Microsoft 这些大厂，后续更开放和社区共建。

Webster、Carroll、Savona 和 Alpert

**React v19.2 发布** —— React 又来了个“大/小”版本，这次新特性包括 `<Activity />` 组件（可以隐藏并恢复子组件的 UI 和内部状态），`useEffectEvent`，以及对 Chrome DevTools 性能分析的优化，现在你能更直观地看到组件的调度和树状结构，非常实用。

🛠  代码与工具

**Coinbase 开源 CDS（Coinbase Design System）** —— Coinbase 团队把自己的 React / React Native 跨平台组件库开源了（Apache 2 协议）。这个组件库有不少有趣的细节，比如 数字滚动组件，还有类似多步骤进度条这种实用基础组件，适合有跨端需求的团队参考——GitHub 地址在这里。

**版本发布：**

- 🗾 **react-map-gl v8.1** —— 用 React 操作 MapboxGL JS 的 API 封装，更加友好。点此体验。

- 🔊 **React Native Audio API v0.9** —— 基于 Web Audio API 的 React Native 音频引擎。

- **json-edit-react v1.29** —— 自动编辑/展示 JSON 或对象数据的组件（在线演示）。

- **React Admin v5.12** —— 做 B2B 前端界面的框架。

- **React Stripe.js v5.2** —— Stripe.js 和 Stripe Elements 的 React 组件更新。

- **Reactylon v3.2** —— 基于 Babylon.js 的 React XR 框架。
