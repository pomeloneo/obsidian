Rome Team

**长按识别二维码查看原文**

https://vitejs.dev/blog/announcing-vite4.html

**长按识别二维码查看原文**

https://lexical.dev/

Meta / Facebook

5. **Bun：一个（仍然）有趣的新 JavaScript 运行时** — Bun 出现在 2022 年夏天，作为一个新的 JavaScript 运行时，不是围绕 V8（像 Node.js 或 Deno 那样），而是围绕 WebKit/Apple 的 JavaScriptCore 构建。它包括自己的捆绑器、转码器、任务运行器和 npm 客户端，但最重要的是拥有比现有选项更多的性能 **改进**，并支持大量的 Node 和 Web API 开箱即用。

**长按识别二维码查看原文**

https://bun.sh/

Jarred Sumner
