# 域名头脑风暴器

此 skill 通过生成有创意的选项并检查哪些确实可以注册，帮助你为项目找到理想域名。

## 何时使用此 Skill

- 开始一个新项目或新公司
- 发布产品或服务
- 创建个人品牌或 portfolio site
- 为现有项目 rebranding
- 为 side project 注册域名
- 当首选域名被占用时寻找可用替代方案

## 此 Skill 做什么

1. **理解你的项目**：分析你正在构建什么以及面向谁
2. **生成有创意的名称**：创建相关、好记的 domain options
3. **检查可用性**：验证多个 TLD 中哪些 domains 确实可用
4. **多个扩展名**：建议 .com、.io、.dev、.ai、.app 等
5. **提供替代方案**：如果首选项被占用，提供变体
6. **品牌洞察**：解释某些名称为什么有效

## 域名命名提示

### 好域名的特征

✓ **短**：最好少于 15 个字符
✓ **好记**：容易回忆和拼写
✓ **可发音**：能在对话中说出来
✓ **描述性**：暗示你做什么
✓ **可品牌化**：足够独特，能脱颖而出
✓ **无连字符**：更容易口头分享

### TLD 指南

- **.com**：通用、可信，适合 businesses
- **.io**：Tech startups、developer tools
- **.dev**：Developer-focused products
- **.ai**：AI/ML products
- **.app**：Mobile 或 web applications
- **.co**：.com 的替代选择
- **.xyz**：现代、创意型 projects
- **.design**：Creative/design agencies
- **.tech**：Technology companies

## 示例工作流

### Startup Launch
1. 描述你的 startup idea
2. 获取跨 TLDs 的 10-15 个 domain suggestions
3. 查看 availability 和 pricing
4. 挑选前 3 个最喜欢的选项
5. 立即注册

### Personal Brand
1. 分享你的姓名和职业
2. 获取变体（firstname.com、firstnamelastname.dev 等）
3. 同时检查 social media handle 可用性
4. 在各平台注册一致品牌

### Product Naming
1. 描述 product 和 target market
2. 获取有创意、可品牌化的名称
3. 检查 trademark conflicts
4. 验证 domain 和 social availability
5. 用目标受众测试名称

## 成功提示

1. **行动要快**：好域名很快会被注册
2. **注册变体**：获取 .com 和 .io 以保护品牌
3. **避免数字**：口头沟通困难
4. **检查 Social Media**：确保 @username 也可用
5. **大声读出来**：测试是否容易发音
6. **检查 Trademarks**：确保没有法律冲突
7. **长期思考**：5 年后它是否仍然合理？

## 价格上下文

- 已占用 domains：如果列出则按市场价
