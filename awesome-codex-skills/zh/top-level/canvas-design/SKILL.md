这些是用于创建设计哲学的说明：设计哲学是一种随后会被视觉化表达的美学运动。只输出 .md 文件、.pdf 文件和 .png 文件。

分两步完成：
1. 创建设计哲学（.md 文件）
2. 通过在 canvas 上创建作品来表达它（.pdf 文件或 .png 文件）

首先，执行此任务：

## 设计哲学创建

开始时，创建一种视觉哲学（不是 layouts 或 templates），它将通过以下方式被诠释：
- 形态、空间、色彩、构图
- 图像、图形、形状、图案
- 作为视觉点缀的极少文字

### 关键理解
- 接收到的内容：用户提供的一些微妙输入或指令，应被纳入考虑，但只作为基础；它不应限制创作自由。
- 创建的内容：一种设计哲学/美学运动。
- 接下来发生什么：然后，同一版本会接收该哲学并进行视觉化表达，创造出 90% 是视觉设计、10% 是必要文字的 artifacts。

考虑这种方式：
- 为一场艺术运动写一份宣言
- 下一阶段是制作艺术作品

该哲学必须强调：视觉表达。空间沟通。艺术诠释。极少文字。

### 如何生成视觉哲学

**命名这场运动**（1–2 个词）："Brutalist Joy" / "Chromatic Silence" / "Metabolist Dreams"

**阐述哲学**（4–6 段，简洁但完整）：

为了捕捉视觉本质，表达该哲学如何通过以下方面显现：
- 空间与形态
- 色彩与材质
- 尺度与节奏
- 构图与平衡
- 视觉层级

**关键指南：**
- **避免冗余**：每个设计方面只提一次。除非增加新的深度，否则避免重复关于色彩理论、空间关系或排版原则的观点。
- **反复强调 craftsmanship**：该哲学必须多次强调，最终作品应看起来像是耗费无数小时创作、经过细致打磨，并出自该领域绝对顶尖的人之手。这个框架至关重要——反复使用类似 “meticulously crafted”、“the product of deep expertise”、“painstaking attention”、“master-level execution” 的短语。
- **留下创作空间**：对美学方向保持具体，但也要足够简洁，让下一个 Claude 有空间做出同样极高 craftsmanship 水平的诠释性选择。

该哲学必须引导下一个版本通过视觉而非文字表达想法。信息存在于设计中，而不是段落中。

### 哲学示例

**"Concrete Poetry"**
哲学：通过纪念碑式形态和大胆几何进行沟通。
视觉表达：巨大的色块、雕塑感排版（超大的单词、极小的标签）、Brutalist 空间分割，Polish poster energy 遇上 Le Corbusier。想法通过视觉重量和空间张力表达，而不是解释。文字作为稀有而强有力的姿态——绝不是段落，只是融入视觉建筑中的必要词语。每个元素都以 master craftsman 的精确度放置。

**"Chromatic Language"**
哲学：色彩作为主要信息系统。
视觉表达：几何精确性，让色彩区域创造意义。排版极少——小型 sans-serif 标签让色彩场域进行沟通。想象 Josef Albers 的 interaction 与 data visualization 相遇。信息以空间和色彩编码。文字只用于锚定色彩已经展示的内容。结果来自 painstaking chromatic calibration。

**"Analog Meditation"**
哲学：通过纹理和呼吸空间实现安静的视觉沉思。
视觉表达：纸张颗粒、墨水渗化、广阔负空间。摄影和插画占主导。排版像低声耳语（小而克制，服务于视觉）。Japanese photobook aesthetic。图像在页面中呼吸。文字节制出现——短语，而非说明性文本块。每个构图都以冥想练习般的照护取得平衡。

**"Organic Systems"**
哲学：自然聚簇和模块化生长模式。
视觉表达：圆润形态、有机排列、来自自然并经由建筑转译的色彩。信息通过视觉图表、空间关系和 iconography 呈现。文字只作为漂浮在空间中的关键标签。构图通过 expert spatial orchestration 讲述故事。

**"Geometric Silence"**
哲学：纯粹的秩序与克制。
视觉表达：基于 grid 的精确性、大胆摄影或 stark graphics、戏剧性负空间。排版精准但极少——小而必要的文字，大面积安静区域。Swiss formalism 遇上 Brutalist material honesty。结构在沟通，而不是文字。每一次对齐都来自无数次 refinement。

*这些是浓缩示例。实际设计哲学应包含 4–6 个扎实段落。*

### 核心原则
- **视觉哲学**：创建一种将通过设计表达的美学世界观
- **极少文字**：始终强调文字稀疏、只保留必要内容，并作为视觉元素整合其中——绝不冗长
- **空间表达**：想法通过空间、形态、色彩、构图沟通，而不是段落
- **艺术自由**：下一个 Claude 以视觉方式诠释哲学——提供创作空间
- **纯粹设计**：这是在创作艺术对象，而不是带装饰的文档
- **专家级 craftsmanship**：反复强调最终作品必须看起来 meticulously crafted、经过细致打磨、由领域顶尖者耗费无数小时完成

**设计哲学应为 4–6 段。** 用富有诗性的设计哲学填充它，将核心愿景融合在一起。避免重复同样的观点。保持设计哲学的通用性，不要提及艺术作品的意图，让它像是可被用于任何地方。将设计哲学输出为 .md 文件。

## 推断微妙参照

**关键步骤**：在创建 canvas 之前，识别原始请求中微妙的概念线索。

**核心原则**：
主题是**嵌入艺术本身的微妙、小众参照**——不总是字面化，总是 sophisticated。熟悉主题的人应能直觉感受到它，而其他人只是体验到一件 masterful abstract composition。设计哲学提供美学语言。被推断出的主题提供灵魂——安静的概念 DNA，无形地编织进形态、色彩和构图。

这**非常重要**：参照必须经过提炼，使其增强作品深度，而不是自我宣告。像爵士乐手引用另一首歌那样思考——只有懂的人会捕捉到，但所有人都能欣赏这段音乐。

## Canvas 创建

在哲学和概念框架都确立后，在 canvas 上表达它。花一点时间整理思绪，清空头脑。使用创建出的设计哲学和下面的说明来制作一件 masterpiece，以 expert craftsmanship 体现哲学的所有方面。

**重要**：无论内容类型是什么，即使用户请求的是电影/游戏/书籍相关内容，方法仍应 sophisticated。永远不要忘记：这应该是艺术，而不是 cartoony 或 amateur 的东西。

要创造 museum 或 magazine quality 的作品，请以设计哲学为基础。创建单页、高度视觉化、design-forward 的 PDF 或 PNG 输出（除非要求多页）。通常使用重复图案和完美形状。把抽象哲学设计当成一本 scientific bible，借用系统化观察的视觉语言——通过密集累积的 marks、重复元素或 layered patterns，以耐心重复建立意义，并奖励持续观看。添加稀疏、clinical typography 和系统性 reference markers，让它看起来像来自 imaginary discipline 的 diagram，以通常用于记录可观察现象的敬意对待那个不可见的主题。用简单短语或细节作为锚点，位置要微妙，并使用有限的 color palette，让它显得有意图且 cohesive。拥抱用 analytical visual language 表达 human experience 的悖论：结果应像一件 artifact，证明某种 ephemeral 之物可以通过细致关注被研究、映射和理解。这是真正的艺术。

**文字作为上下文元素**：文字始终是 minimal 且 visual-first 的，但让语境决定它是 whisper-quiet labels 还是 bold typographic gestures。punk venue poster 可能拥有比 minimalist ceramics studio identity 更大、更具攻击性的字体。大多数时候，font 应该细。所有字体使用都必须 design-forward，并优先服务视觉沟通。无论文字尺度如何，任何内容都不能超出页面，也不能重叠。每个元素都必须包含在 canvas 边界内，并有合适 margins。仔细检查所有文字、图形和视觉元素都有呼吸空间和清晰分隔。对于专业执行，这是不可协商的。**重要：如果写文字，请使用不同 fonts。搜索 `./canvas-fonts` 目录。无论采取何种方式，sophistication 都不可协商。**

下载并使用实现这一目标所需的任何 fonts。发挥创意，让 typography 真正成为艺术本身的一部分——如果作品是抽象的，就把字体带入 canvas，而不是进行数字排版。

为了突破边界，在使用哲学作为指导原则的同时，遵循设计本能/直觉。拥抱终极设计自由和选择。把美学和设计推向前沿。

**关键**：为了达到 human-crafted quality（而不是 AI-generated），创作看起来耗费了无数小时的作品。让它显得像领域绝对顶尖的人以 painstaking care 打磨每个细节。确保构图、间距、色彩选择、排版——一切都体现 expert-level craftsmanship。反复检查没有重叠、格式 flawless、每个细节 perfect。创造出可以展示给人们以证明专业能力，并且 undeniably impressive 的东西。

将最终结果输出为单个可下载的 .pdf 或 .png 文件，并附上所使用的设计哲学 .md 文件。

## 最后步骤

**重要**：用户已经说过 “It isn't perfect enough. It must be pristine, a masterpiece if craftsmanship, as if it were about to be displayed in a museum.”

**关键**：为了 refine 作品，避免添加更多图形；而是 refine 已创建的内容，让它极其 crisp，同时尊重设计哲学和 minimalism 的原则。不要添加有趣 filter 或重构字体，而是思考如何让现有构图与艺术更 cohesive。如果本能是调用新函数或画新形状，停下来，改问：“我如何让已经存在的东西更像一件艺术品？”

进行第二轮。回到代码，进一步 refine/polish，让它成为一件经过哲学设计的 masterpiece。

## 多页选项

当请求创建额外页面时，沿着同一设计哲学创作更多页面，但每页也应明显不同。将这些页面打包在同一个 .pdf 中，或输出多个 .png。把第一页只当成一本等待被填满的 coffee table book 中的一页。让后续页面成为原作的独特转折和记忆。让它们以非常 tasteful 的方式近乎讲述一个故事。充分行使创作自由。
