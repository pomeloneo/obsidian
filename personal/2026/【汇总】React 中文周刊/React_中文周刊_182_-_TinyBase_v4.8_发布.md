**版本发布：**

- **Jotai v2.8** - 为 React 提供简单、灵活的状态管理。引入了 `atomWithLazy` 和一个实验性的在 `jotai/experimental` 中的 store 实现。

- **React on Rails v14.0** – 一种将 React 与 Ruby on Rails 应用集成的方式。

- **React Native Skia v1.1** – 为 React Native 提供高性能的 2D 图形。

- **React Testing Library v14.3** – 增加了对 React 19 Canary 的支持。

- **P5-wrapper/react v4.4** – 将 P5.js 的草图集成到 React 应用程序中，这是 演示。

- **react-medium-image-zoom v5.2** – 受 Medium.com 启发的图片组件。

- **react-native-bootsplash v5.5.2** – 在启动过程中显示一个启动页面。

- **React Stripe.js v2.7** – 为 Stripe.js 和 Stripe Elements 提供的组件。

- **🏷 Tagify v4.23** – 轻量级的标签输入组件。

- **Redwoodv7.3** – GraphQL 驱动的 React 框架。
