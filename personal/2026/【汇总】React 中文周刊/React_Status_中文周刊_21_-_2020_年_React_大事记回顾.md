## 📅 2020 React 每月大事记

**一月：** Dan Abramov 回顾了他过去 10 年的工作。TypeScript 团队使用 Gatsby 重构 TypeScript 文档。Styled Components v5 发布。

**二月：** React Native 崛起采用了全新的域名：reactnative.dev。CRA 官方的 Redux 模板 1.0 版本发布。React Query 1.0 问世。

**三月：** RedwoodJS JAMStack 方案出现。冠状病毒开始肆虐，扰乱了 React 团队及社区的相关计划。

**四月：** Preact 10.4.0 发布。ZEIT 更名为 Vercel。

**五月：** Next.js 9.4 发布（首要特性就是 fast-refresh 和增量构建）。Recoil 发布。Gatsby 民间筹款。

**六月：** React 核心团队加入 Facebook 员工罢工活动，声援黑人社区。MDN 发布了 React 教程。

**七月：** Adobe 推出了 UI 组件库 React Spectrum。微软推出了商业报告分析工具 React Power BI。Next.js 9.5 发布。

**八月：** Facebook 透露了很多关于 Fackebook 内部如何使用 React 的信息。React 17.0 RC-1 发布，无新特性。

**九月：** Kent C Dodds 编写了一封给 React 的情书。从中可以了解更多有关 React 全新的 JSX 转换的相关信息。Vue 3.0 发布。

**十月：** webpack 发布了 5.0 版本。React 17.0 正式发布。Create React App 紧随其后发布了 4.0 版本，同时发布的还有 Next.js 10。

**十一月：** 全新的 Redux 基础知识教程发布。英国广播公司 BBC 对外展示了其基于 React 的工作。

**十二月：** React 圣诞特辑 开启更新。

## 🛠 2020 年以来顶级的代码与工具

**Recoil：React 试验性状态管理库** — 有趣的是，不仅在于它来自 Facebook 团队，更重要的是还有包含 23 分钟的精彩演讲，另外它还遵循了 React 的最新标准。

原文链接：https://recoiljs.org/

Facebook

**RedwoodJS：符合 JAMstack 规范的全栈应用框架** — 一个非常优秀的框架，融合了 React, GraphQL 和 Prisma2，还有许多其他框架。值得注意的是，它来自于 Github 最初创始人之一，Gravatar 的创建者 Tom Preston-Werner。我们预计 2021 年会有更多符合 JAMStack 规范的全栈框架出现。

原文链接：https://redwoodjs.com/

RedwoodJS

**Reactochart：由 Spotify 推出的图表组件库** — 过去我们发现了很多图表库，但是看到像 Spotify 这样的知名品牌提供的并不多见（而且这个组件库已经成熟并且存在了一段时间了）。它的文档还是很不错的，每种类型的图表都有很不错的案例，比如折线图，柱状图，饼图，直方图，热力图等。

仓库地址：https://github.com/spotify/reactochart

Spotify

React Query 3：获取、缓存和更新数据的 Hooks — React Query 昨天刚刚发布了 3.2.0 版本。它可以帮助你缓存服务器状态，并且将缓存状态与全局状态分离，另外还允许你异步读取和更新所有状态。

仓库地址：https://github.com/tannerlinsley/react-query

Tanner Linsley

React Hook Form 6：简单、可扩展的 React 表单校验 Hook — RHF 已经更好的适配了 TypeScript，包的尺寸更小了，还有很多重构和改进。如果你正在构建新的表单或者重构历史表单，RHF 非常值得一看。查看 GitHub 项目了解更多。本周刚刚发布了一个新版本。

原文链接：https://react-hook-form.com/

Blue Bill

Craft.js：一个用于构建拖拽页面编辑器的 React 框架 — 将项目的引导页以文本编辑器的形式展示是一个大胆的举动，但是我们觉得这很酷。:-)

原文链接：https://craft.js.org/

Prev Wong
