# 一、开发环境准备：打造高效稳定的 AI 开发工作台

在开始编码之前，我们需要先为项目准备好一个稳定、隔离且高效的开发环境。这不仅能避免依赖冲突，还能提升团队协作效率，是专业开发的第一课。

## 1.1 技术栈选型：为什么是 Python + FastAPI？

在大模型应用开发中，Python 凭借其丰富的 AI 生态（如 LangChain、LlamaIndex、Hugging Face 等）和简洁的语法，已成为事实上的标准语言。我们选择 Python 3.10+ 作为主语言，确保兼容最新库版本。

后端框架我们选用 FastAPI，原因如下：

高性能：基于 ASGI（异步服务器网关接口），支持异步请求处理，适合大模型的高延迟 IO 场景；

自动文档生成：集成 Swagger UI，便于接口调试与团队协作；

类型提示友好：与 Pydantic 深度集成，提升代码可读性与健壮性；

轻量灵活：无过多约定，适合快速迭代的 AI 应用开发。

替代方案：如果你更熟悉 Flask 或 Django，也可使用，但建议优先尝试 FastAPI 以体验异步优势。

## 1.2 环境管理：使用 venv 或 conda 创建虚拟环境

为了避免不同项目间的依赖冲突，我们必须使用虚拟环境。

使用 venv（推荐新手）：

```bash
python -m venv venv
```

source venv/bin/activate

venv\Scripts\activate

使用 conda（适合数据科学背景）：

conda create -n emotion-chat python=3.10

conda activate emotion-chat

激活后，终端前缀应显示环境名，如 venv 或 emotion-chat，表示环境已生效。

## 1.3 包管理：pip 与 requirements.txt

我们使用 pip 安装依赖，并通过 requirements.txt 锁定版本，确保环境一致性。

初始化项目后，创建空文件：

## 2.1 路由层 (Router Layer)

路由层负责 API 接口定义和请求分发。

backend/routers/

```text
├── chat.py - 聊天接口
├── memory.py - 记忆管理接口
├── feedback.py - 用户反馈接口
├── evaluation.py - 评估系统接口
└── agent_router.py - Agent智能接口
```

## 2.2 服务层 (Service Layer)

服务层用于业务逻辑封装和编排。

backend/services/

```text
├── chat_service.py - 聊天服务
├── memory_service.py - 记忆服务
├── context_service.py - 上下文服务
└── agent_service.py - Agent服务（可选）
```

## 2.3 核心层 (Core/Module Layer)

核心功能模块如下所示。

backend/modules/

```text
├── agent/ - Agent智能核心
│ ├── core/ - Agent Core, Planner, Tool Caller等
│ ├── services/ - Agent服务实现
│ └── routers/ - Agent路由
├── rag/ - RAG知识库系统
│ ├── core/ - 知识库管理、检索
│ ├── services/ - RAG服务
│ └── routers/ - RAG路由
└── llm/ - LLM模型封装
└── qwen_client.py - 通义千问客户端
```

## 3.2 配置加载：config.py

之后来完成配置加载。

```python
import os
from pathlib import Path
from dotenv import load_dotenv
env_path = Path(__file__).parent / 'config.env'
load_dotenv(env_path)
PROJECT_ROOT = os.getenv('PROJECT_ROOT', str(Path(__file__).parent))
class Config:
OPENAI_API_KEY = os.getenv("OPENAI_API_KEY")
LANGCHAIN_TRACING_V2 = os.getenv("LANGCHAIN_TRACING_V2", "false").lower() == "true"
LANGCHAIN_API_KEY = os.getenv("LANGCHAIN_API_KEY")
DATABASE_URL = os.getenv("DATABASE_URL")
MYSQL_HOST = os.getenv("MYSQL_HOST", "localhost")
MYSQL_PORT = int(os.getenv("MYSQL_PORT", "3306"))
MYSQL_USER = os.getenv("MYSQL_USER", "root")
MYSQL_PASSWORD = os.getenv("MYSQL_PASSWORD")
MYSQL_DATABASE = os.getenv("MYSQL_DATABASE", "emotional_chat")
CHROMA_PERSIST_DIRECTORY = os.getenv("CHROMA_PERSIST_DIRECTORY",
os.path.join(PROJECT_ROOT, "chroma_db"))
DEFAULT_MODEL = os.getenv("DEFAULT_MODEL", "gpt-4")
TEMPERATURE = float(os.getenv("TEMPERATURE", "0.7"))
MAX_TOKENS = int(os.getenv("MAX_TOKENS", "1000"))
HOST = os.getenv("HOST", "0.0.0.0")
PORT = int(os.getenv("PORT", "8000"))
DEBUG = os.getenv("DEBUG", "false").lower() == "true"
```
